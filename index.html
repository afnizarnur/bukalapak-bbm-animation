Bounce

Usage bounce animation for example when user adds item to cart.

            
.bounce {
    animation: bounce 1s infinite cubic-bezier(0.075, 0.82, 0.165, 1);
}

@keyframes bounce {
    0%, 20%, 50%, 80%, to {
        transform: translateY(0);
    }

    40% {
        transform: translateY(-15px);
    }

    60% {
        transform: translateY(-5px);
    }
}
