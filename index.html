Bounce

Penggunaan bounce contohnya ketika user menambahkan barang ke cart.

            
                .bounce {
                    animation: bounce 1s infinite cubic-bezier(0.075, 0.82, 0.165, 1);
                }

                @keyframes bounce {
                    0%, 20%, 50%, 80%, to {
                        transform: translateY(0);
                    }
                
                    40% {
                        transform: translateY(-15px);
                    }
                
                    60% {
                        transform: translateY(-5px);
                    }
                }
Slide Left

Penggunaan slide left contohnya ketika memunculkan page overlay atau page diatasnya page.

            
                .slideLeft {
                    animation: slideLeft 0.8s infinite cubic-bezier(0.6, 0.01, 0.13, 1.01);
                }
                
                @keyframes slideLeft {
                    0% {
                        transform: translateX(-50px);
                    }
                    100% {
                        transform: translateX(50px);
                    }
                }
